 Project Overview

This project is created as part of the Digital Literacy course at VIT Bhopal University. As a Student Digital Ambassador, the objective of this project is to create awareness among students about digital tools, online communication, cybersecurity, and building a professional digital presence.

The project consists of five tasks that focus on different aspects of digital literacy including infographic creation, portfolio building, coding platforms, email etiquette, and cybercrime awareness.

 Task Summary
Task 1 – Digital Literacy Infographic

An infographic was created using Canva explaining key concepts of digital literacy such as safe internet usage, useful tools for students, and maintaining a professional online presence.

 Task 2 – Digital Portfolio

Profiles were created and updated on:

GitHub
LinkedIn
Kaggle

These platforms help in building a professional identity and showcasing technical skills.

 Task 3 – Platforms Exploration
Part A – Coding Platform

A beginner-level problem was solved on HackerRank to understand coding practice platforms.

Part B – Google Form

A Digital Literacy Awareness Quiz with 5 questions was created using Google Forms.

 Task 4 – Email Etiquette

Two professional emails were written:

Also, a checklist of Do’s and Don’ts for responsible social media usage was created.

 Task 5 – Cybercrime Awareness

A case study on phishing was created explaining how cyber attacks happen and their consequences.

A prevention checklist was also developed to help students stay safe online.
